# Soft Minimalist Toast Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement global toast notifications for all mutation actions using `shadcn/ui` sonner, styled with a custom "Soft Minimalist" aesthetic (rounded pills, soft shadows, clean typography).

**Architecture:**

- Install `sonner` via shadcn CLI.
- Override the default square `toastOptions` in `src/components/ui/sonner.tsx` to use rounded, pill-like shapes with a soft shadow.
- Mount the `<Toaster />` globally in `src/routes/__root.tsx`.
- Add `toast.success` and `toast.error` to the `onSuccess` and `onError` callbacks of all user and role mutation hooks.

**Tech Stack:** React, Tailwind CSS, shadcn/ui, Sonner, TanStack Query.

### Task 3: Integrate Toasts into User Mutations

**Files:**

- Modify: `src/features/users/hooks/use-create-user.ts`
- Modify: `src/features/users/hooks/use-edit-user.ts`

- [ ] **Step 1: Update `use-create-user.ts`**

Import `toast` from `'sonner'` and add `toast.success` and `toast.error` handlers to `useMutation`.

```tsx
import { useMutation, useQueryClient } from "@tanstack/react-query";
import { toast } from "sonner";
import { createUser } from "../api";
import type { CreateUserPayload } from "../types";

export function useCreateUser() {
  const queryClient = useQueryClient();

  return useMutation({
    mutationFn: (payload: CreateUserPayload) => createUser(payload),
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: ["users"] });
      toast.success("User created successfully");
    },
    onError: (error: any) => {
      toast.error(error?.response?.data?.message || "Failed to create user");
    },
  });
}
```

- [ ] **Step 2: Update `use-edit-user.ts`**

Import `toast` and update the hook:

```tsx
import { useMutation, useQueryClient } from "@tanstack/react-query";
import { toast } from "sonner";
import { editUser } from "../api";
import type { EditUserPayload } from "../types";

export function useEditUser(userId: string) {
  const queryClient = useQueryClient();

  return useMutation({
    mutationFn: (payload: EditUserPayload) => editUser(userId, payload),
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: ["users"] });
      queryClient.invalidateQueries({ queryKey: ["user", userId] });
      toast.success("User updated successfully");
    },
    onError: (error: any) => {
      toast.error(error?.response?.data?.message || "Failed to update user");
    },
  });
}
```
